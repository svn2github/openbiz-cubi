---------------------------------
Steps to start after installation
---------------------------------
1. Setup xampp 
Click Start Menu -> All Programs -> Openbiz-Cubi -> Setup Xampp
A command window is opened to execute xampp servers setup

2. Start xampp control panel
Click Start Menu -> All Programs -> Openbiz-Cubi -> Start Xampp Control Panel
Xampp control panel will be launched for user to manage apache and mysql services

3. Launch Openbiz Cubi
Click Start Menu -> All Programs -> Openbiz-Cubi -> Launch Openbiz Cubi
A browser will start to load local Openbiz Cubi url. E.g. http://localhost/cubi

---------------------------
Go to Openbiz Cubi website
---------------------------
Click Start Menu -> All Programs -> Openbiz-Cubi -> Website
A browser will start to load official openbiz website http://www.openbiz.me
