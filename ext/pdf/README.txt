Pdf Cubi extension provides support for pdf generation from web page.

- This extension uses mpdf library http://www.mpdf1.com/mpdf/

- In order to save source space, mpdf fonts folder are not included in the source tree. If you need to use special ture type fonts, you can download them from
 1. from mpdf site.
 2. from phpopenbiz sourceforge svn http://bigchef.svn.sourceforge.net/viewvc/bigchef/branches/3.0/cubi/modules/pdf/lib/mpdf50/ttfonts/

Then copy mpdf/ttfonts folder to pdf/lib/mpdf50/ttfonts.

- You are free to replace mpdf50 source with the latest source from mpdf official site.
