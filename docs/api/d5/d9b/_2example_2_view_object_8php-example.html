/example/ViewObject.php

Get the openbiz view object by object name. It's functional same as getObject() method, just this method can return more eclipse friendly result, it can support IDE's code auto completaion.

Parameters
string$objectNameobject name
Returns
EasyView if the return object is a EasyView then return, or return null
<?php
//include openbiz initail script
require_once dirname(dirname(__FILE__)).'/bin/app_init.php';
$objectName = "system.view.UserListView";
//get the view object instance
$userView = BizSystem::getViewObject($objectName);
//render the view to a html string
$viewHTML = $userView->render();
//output the html string
echo $viewHTML;
?>
